Owner console

This page is not linked from the public FairShare app. Access is enforced by a secret only you store in .env on the server. Use npm start and open this file from the same origin (e.g. http://localhost:3840/fs-owner-console.html).

Anyone with this URL and your ADMIN_PANEL_SECRET can see these numbers. Do not bookmark the secret in a shared machine; prefer entering it each time or use a private browser profile.
